# Akram Naamane - Developer Portfolio

A modern, minimal, and responsive developer portfolio built with **HTML, CSS, and JavaScript**.

This portfolio is designed to showcase my software development projects, technical skills, and GitHub profile in a professional format suitable for recruiters and internship applications.

## Features

- Dark modern UI
- Responsive layout for desktop, tablet, and mobile
- Smooth scrolling navigation
- Animated section reveal on scroll
- Typing effect in the hero section
- Project cards with hover effects
- Easy deployment on GitHub Pages

## Technologies Used

- HTML5
- CSS3
- JavaScript
